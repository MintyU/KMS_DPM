# KMS_StarSimul v1.0

- Project using C++

Korean MapleStory Server (이하 KMS)의 스타포스 시뮬레이션 및 기댓값 계산 프로그램입니다.

# 구현 내용

- 스타포스 확률, 가격 데이터 저장 /t (O)
- 강화할 아이템 레벨 입력 구현     (O)
- 목표 스타포스 단계 입력 구현    (O)
- 스타캐치 적용 유무 입력 구현    (O)
- MVP 등급 할인 차등 적용 구현    (O)
- 장비파괴 및 복구 기능 구현     (O)
- 2회 연속 등급 하락 시 찬스타임 구현 (O)
- 총 사용 메소와 파괴 횟수 표시 구현  (O)
- 입력 예외 처리 구현             (O)
